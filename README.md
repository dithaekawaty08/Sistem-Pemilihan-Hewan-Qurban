# Sistem-Pemilihan-Hewan-Qurban
Program Pemilihan Hewan Qurban berbasis Java. Program ini merupakan tugas akhir ( UAS ) dalam mata kuliah pemrograman berbasis objek dengan menerapkan berbagai konsep OOP seperti Encapsulation, Polymorphism, Inheritance dll. Program ini dapat membantu pengelolaan data hewan secara terstuktur dan efisien dengan memungkinkan pengguna dapat melihat dan membeli hewan qurban secara otomatis sedangkan untuk admin dapat mengubah dan menambahkan data hewan qurban.

# Fitur dalam program
- Menampilkan list harga hewan qurban
- Melakukan pembelian hewan qurban
- Menambah data hewan qurban
- Memilih hewan dan metode pembayaran
# Teknologi yang digunakan
- Java
- Visual Studio Code ( JDK 25.0.2 )
# Stuktur Program Utama
- MainQurban.java = Program utama
- AdminMenu. java = Menu Admin
- HewanQurban.java = Class Induk
- Sapi.Java = Contoh subclass
- Pembayaran = Class untuk pembayaran
